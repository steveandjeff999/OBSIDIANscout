Upload Excel File

Import scouting data from an Excel file (.xlsx or .xls format). The Excel file should have the following columns:

Required columns:
  • match_number - The match number
  • team_number - The team number
Optional columns:
  • match_type - Type of match (defaults to "Qualification")
  • scout_name - Name of the scout (defaults to "Excel Import")
  • alliance - "red" or "blue" (defaults to "unknown")
  • event_name - Name of the event (defaults to "Unknown Event")
Plus any scoring element columns as defined in your game configuration.
Only .xlsx and .xls files are supported.
Note: Existing data for the same team and match will be updated.
